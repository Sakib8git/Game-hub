---

## Purpose

- Provide a professional-grade user experience with Google login, password reset, and dynamic navigation
- Supporting game developers
- Firebase Authentication with protected routes
- Implement client validation, routing, and animation

---

## Key Features

- Design: Gradient backgrounds, responsive layout, hover/tap animations
- Game Pages: All Games, Popular Games, Game Details with ratings and download links
- Routing: Protected routes, redirect after login, dynamic detail pages
- Authentication: Email/password login, Gogle sign-in, password reset
- Validation: RegEx-based client-side form validation
- Animation: Framer Motion for hover, entry, and page transitions
